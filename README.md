# easythreed-k9-official-firmware-
this is the official firmware for the easythreed k9,you can use it if you had any problem with your k9 printer 
Guide for updating 
1-copy the files to the sd card(except reed.md file)
2-put the sd card in the easythreed k9 printer 
3-turn it on
4-wait for 1 minute
after that you should see a file end with .cur 
and thats it
